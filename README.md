## ✨ 概要 (Overview)

**m4 task** は「シンプルさ」と「滑らかさ」に焦点を当てて設計されています。不要な複雑さを削ぎ落とし、集中できるクリーンなワークスペースを提供しつつ、あなただけのアプリにするための深いカスタマイズオプションを備えています。

## 🎨 主な機能 (Features)

- 💎 **ミニマルなUI**: 読みやすさと集中力を高めるために設計されたクリーンなワークスペースと、滑らかなアニメーション。
- 🖌️ **自由なカスタマイズ**: 
  - アクセントカラーを好きなHexコードに変更可能。
  - お気に入りの画像を背景に設定し、ぼかし具合を調整可能。
  - フローティングアクションボタン（FAB）の配置を好みの場所に移動可能。
- 📱 **インタラクティブなウィジェット**: ホーム画面から直接タスクの追加、確認、完了ができるレスポンシブなAndroidウィジェット。
- 📊 **統計データ**: 完了したタスクの総数と、直近24時間のタスク完了率を確認可能。
- 📦 **バックアップ＆復元**: タスク、設定、背景画像をひとつのZIPファイルとして簡単にバックアップ可能。
- 🌙 **ダークモード対応**: 目に優しい高コントラストなディープグレーベース。
- 🌍 **多言語対応**: 日本語、英語、簡体字中国語、繁体字中国語をサポート。

## 🚀 インストール方法 (Installation)

最新のAPKファイルは、[Releases](https://github.com/masaki-09/m4sak1-task-app/releases) ページから直接ダウンロードしてインストールできます。

## 🛠️ 技術スタック (Technical Stack)

- **UIフレームワーク**: Jetpack Compose (Material 3) & Glance (Widgets)
- **データベース**: Room Persistence Library
- **非同期処理**: Kotlin Coroutines & Flow
- **データシリアライズ**: Kotlinx Serialization
- **アーキテクチャ**: MVVM

## 🧑‍💻 開発環境 (Development)

1. リポジトリをクローンします:  
```bash
git clone https://github.com/masaki-09/m4sak1-task-app.git  
```
2. **Android Studio** でプロジェクトを開きます。
3. Gradle同期を行い、`:app` モジュールを実行します。
