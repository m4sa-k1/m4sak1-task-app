## ✨ Overview

**m4 task** is designed with a focus on simplicity and fluidity. We stripped away unnecessary complexities to give you a clean workspace, while providing deep customization options so you can make the app truly yours. 

## 🎨 Features

- 💎 **Minimalist UI**: A clean workspace designed for readability and focus, featuring fluid animations.
- 🖌️ **Unbound Customization**: 
  - Change the accent color to any Hex code.
  - Set your favorite image as the background with adjustable blur.
  - Move the Floating Action Button (FAB) wherever it feels most comfortable.
- 📱 **Interactive Widget**: Add, view, and complete tasks directly from your home screen with a responsive Android widget.
- 📊 **Statistics**: Track your total completed tasks and monitor your 24-hour completion rate.
- 📦 **Backup & Restore**: Easily safeguard your tasks, settings, and backgrounds in a single ZIP file.
- 🌙 **Dark Mode**: High-contrast, deep gray base for eye comfort.
- 🌍 **Localization**: Supports English, Japanese (日本語), Simplified Chinese (简体中文), and Traditional Chinese (繁體中文).

## 🚀 Installation

You can download the latest APK directly from the [Releases](https://github.com/masaki-09/m4sak1-task-app/releases) section.

## 🛠️ Technical Stack

- **UI Framework**: Jetpack Compose (Material 3) & Glance (Widgets)
- **Database**: Room Persistence Library
- **Concurrency**: Kotlin Coroutines & Flow
- **Data Serialization**: Kotlinx Serialization
- **Architecture**: MVVM

## 🧑‍💻 Development

1. Clone the repository:  
```  
git clone https://github.com/masaki-09/m4sak1-task-app.git  
```
2. Open the project in **Android Studio**.
3. Sync Gradle and run the `:app` module.
